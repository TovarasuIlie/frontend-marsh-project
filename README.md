# 🛡️ ApexBank Terminal | Frontend Interface

### High-Precision Financial Management System
**ApexBank Device Management System** is a high-precision, administrative terminal designed for the secure tracking and allocation of corporate hardware assets. Built with a focus on "Fintech-grade" performance, it provides a centralized registry for workstations, mobile units, and tablets across global sectors..

## 🚀 Quick Start Protocol

### PREREQUISITES
Ensure your local environment meets the minimum security clearance:
* **Node.js**: v18.13.0 or higher
* **NPM**: v9.0.0 or higher
* **Angular CLI**: v17.0.0+

### 1. ENVIRONMENT SYNC
Clone the repository and enter the directory:
```bash
git clone https://github.com/TovarasuIlie/frontend-marsh-project
cd frontend-marsh-project
```
### 2. DEPENDENCY INJECTION
Install the required modules for the terminal interface.
```bash
npm install
```

### 3. SYSTEM LAUNCH
Initialize the local development server with hot-reload capabilities.
```bash
ng serve
```
Once the compilation is [SUCCESSFUL], navigate to the local terminal gateway:
```bash
http://localhost:4200/
```
